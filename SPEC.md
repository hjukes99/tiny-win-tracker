# Tiny Win Tracker — SPEC

## App Summary
Tiny Win Tracker is a minimal web app to log small daily wins (e.g., "sent proposal", "worked out", "shipped bugfix") and review momentum over the last 7 days.

## Target User
- Solo builders, students, and operators who want lightweight progress tracking without heavy journaling.

## MVP (24–48h)
1. Add a win (short text + timestamp).
2. List wins for today.
3. Show a 7-day count summary.
4. Persist wins to local JSON storage.
5. Simple REST API + static frontend shell.
6. Basic validation and smoke tests.

## Non-Goals (for now)
- User authentication
- Cloud sync
- Team collaboration
- Mobile apps
